# 🔐 Network Sniffer (CodeAlpha Internship)

A simple Python-based network sniffer that captures and analyzes network packets using Scapy.

---

## 🚀 Features
- Captures live network traffic
- Displays source and destination IP addresses
- Identifies protocols (TCP/UDP)
- Helps understand packet structure

---

## 🛠 Tech Used
- Python
- Scapy

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
sudo python sniffer.py

```
